Hexo+GitHub=博客

发表于 2016-09-16   |   分类于 git   |  

通过Hexo框架和GitHub Page服务搭建免费的个人博客。

搭建

1.提前准备

  • 安装Git
  • 安装Node
  • 拥有GitHub账号

SSH

配置

Git Bash中运行

复制id_rsa.pub文件中的内容，粘贴在GitHub的SSH key设置。

id_rsa.pub的路径： C:\Users\windows_username.ssh\id_rsa.pub

验证

Git Bash中运行

验证信息如下，即为成功配置SSH。

2.搭建本地博客（安装Hexo）

  • 打开Git bash

  • 输入npm命令

  • 新建hexo文件夹（D:/hexo）

  • 在hexo文件夹中，单击右键，选择Git Bash

  • 输入命令

  • 安装依赖包

  • 发布

  • 本地服务器（默认 localhost:4000）

至此，博客已经建立成功（本地），

3.新建GitHub库

  • 打开你的GitHub，新建版本库，名称为 GitHubID.github.io（GitHubID是你自己的id,必须按照这个格式来）

4.部署

  • 打开 D:\hexo\_config.yml（用sublime打开）
  • 寻找Deployment(Ctrl+F)，替换此处代码，其中repo后面有两个‘ljw-bigtail’，第一个是你的GitHubID，第二个是刚才设置的库名
